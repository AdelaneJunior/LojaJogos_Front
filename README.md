# Barraca de Jogos do Zebu

## Projeto realizado durante a disciplina de Programação Web I - Sistemas de Informação UEG
 
  Durante esse projeto tive meu primeiro contato com JavaScript e o framework Angular. A ideia do projeto é adentrar o mundo da programação para a web utilizando conceitos MVC e frameworks consolidados no mercado de programação atual.
  
  Link para o repositorio com codigos back-end: https://github.com/AdelaneJunior/LojaJogos-Back
